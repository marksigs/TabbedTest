ExperianTestClient is a console test program for Experian.  
The 'RSO' writes its request/reply to the console window.
In addition, it outputs a success/failure message at the end.

Built from $EGG/DEV/RPRSO/EXPERIANTESTCLIENT.
Requires the RSO environment variables and files to be set up (i.e. the generic client).
Requires the TOPEND DLLs on the path.
