Omiga POS ActiveX Control Installation

Navigate directly to this page to install onto the desktop the ActiveX controls required by the Omiga POS application.

To be able to install the ActiveX controls, you must have the correct security settings in Internet Explorer.

  1. Run Internet Explorer.
  2. Click on Tools on the menu, and select Internet Options.
  3. On the Security tab, select Local internet.
  4. Click Custom Level.
  5. Ensure the following (all under ActiveX controls and plug-ins) are set:
    • Download signed ActiveX controls: Enable.
    • Run ActiveX controls and plug-ins: Enable.
    • Script ActiveX controls marked safe for scripting: Enable.
  6. Click OK twice to exit out of the options.

To install the ActiveX controls:

  1. Navigate to this page by running Internet Explorer, and entering http://server/OmigaWebSite/cabs/install.htm into the Address field, where server is the name of the Omiga web server, and OmigaWebSite is the name of the Omiga web site on that server (normally called OmigaPOS).
  2. For each ActiveX control being installed, you should see a Security Warning asking if you want to install and run a cab file distributed by Marlborough Stirling; answer Yes to each of the warnings.
  3. Wait for Done to appear in the Internet Explorer status bar (at the bottom left of the window).
  4. If you are prompted to restart your computer, answer Yes. Once the computer has restarted, go back to step 2 until no more ActiveX controls are downloaded.

To verify the ActiveX controls have been successfully downloaded:

  1. Run Internet Explorer.
  2. Click on Tools on the menu, and select Internet Options.
  3. In the Temporary Internet Files section, click Settings.
  4. On the Settings dialog, click View Objects.
  5. In the Program Files column, there should be an entry, with a Status of Installed, for each of the downloaded ActiveX controls. This list is currently:
    • axword.axwordclass: used by Omiga MX and Omiga POS (Retail).
    • PCAttributesBO Class: used by Omiga MX and Omiga POS (Retail).
